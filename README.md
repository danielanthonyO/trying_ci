# Tietokonelepuski CRM

A web-based CRM and service management system for a computer repair business (Leppävaaran Tietokonehuolto).

---

## Project Overview

This project aims to build a responsive web application for managing customers, devices, and repair services.  
The system will support service tracking, cost estimates, status updates, authentication, and role-based access control.

The application will be available in Finnish and work on both desktop and mobile devices.

---

## Project Goals

- Customer register (individuals & companies)
- Service tickets for devices and repair work
- Status tracking (e.g., received, diagnostics, waiting approval, in repair, ready, delivered)
- Cost estimate workflow (send estimate, customer approves/declines)
- History/log of changes
- Role-based access + authentication
- Responsive UI (mobile & desktop)
- Finnish language user interface
